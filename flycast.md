# Flycast
Sega Dreamcast emulator

- [GitHub repository](https://github.com/flyinghead/flycast)

## Installation
Source: [Google Play Store](https://play.google.com/store/apps/details?id=com.flycast.emulator)

## Configuration

*Settings -> General -> Content Location*
Location: SD Card
Directory: `/PocketGames/roms/dreamcast`

*Settings -> Video -> Graphics API
Default: OpenGL

*Settings -> Video -> Internal Resolution*
1600 x 1200 (x2.5)
