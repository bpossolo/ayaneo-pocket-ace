## ROM File Naming Convention
Use this naming convention for ROM files so that emulators/frontends can retrieve metadata (cover art, preview videos, etc) for ROM games.

[Official No-Intro Naming Convention](
https://datomatic.no-intro.org/stuff/The%20Official%20No-Intro%20Convention%20(20071030).pdf)
