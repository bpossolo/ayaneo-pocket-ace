# PPSSPP
Sony PlayStation Portable (PSP) Emulator

## Installation
Source: [Google Play Store](https://play.google.com/store/apps/details?id=org.ppsspp.ppsspp)

## Configuration
Location: SD Card  
System Folder Parent Directory: `/PocketGames/app-data`  
System Folder: `/PocketGames/app-data/PSP`  
Games Folder: `/PocketGames/roms/psp`  

**_Settings -> Graphics -> Backend_**  
Default: Vulkan  

**_Settings -> Graphics -> Rendering Resolution_**  
Default: 2x  
Setting: 4x PSP (1080p)  

**_Settings -> Graphics -> Show FPS Counter_**  
Default: disabled  
Setting: enabled  

### Controls
**_Settings -> Controls -> On-screen touch controls_**  
Default: enabled  
Setting: disabled  

**_Button Mappings_**
| Pocket Ace | PSP |
| --- | --- |
| A | Circle |
| B | Cross |
| Y | Square |
| X | Triangle |
