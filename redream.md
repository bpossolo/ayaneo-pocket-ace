# Redream
Sega Dreamcast emulator

Redream used to be an open-source project (hosted on GitHub) but the developer decided to monetize the application and moved it into a private repository on GitLab.

- [Official Site](https://redream.io)
- [GitLab Issue Tracker](https://gitlab.com/inolen/redream)
- [GitHub repository - defunct](https://github.com/inolen/redream)
